

## Command

#### Setup
```bash
npm install
```
#### Develop
```bash
# using nodemon
npm start
```

#### Test
```bash
npm test
```


## Dependencies

- Watcher and hot-reload: [nodemon](http://nodemon.io/)
- Test:
    + [mocha](https://mochajs.org/)
    + [should](https://github.com/shouldjs/should.js)
    + [supertest](https://github.com/visionmedia/supertest)
